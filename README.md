# Java project: German Article Practicing App

This project is about get acquainted with Swing
(ActionListener, MouseListener, JFrame, JInternalFrame, JButton, JTextArea, Color, Border, Image ..)

Result is a Java application for Windows to help learning German articles.
1. write the G.Word & article
2. write the H.word
3. choose the right article (RadioButtons)
You will get always immediately the right answers.

Score table  (in encrypted data files :-)

2. the Last Rounds
